Deskripsi: 
Sistem ini digunakan untuk memproses pengiriman paket melalui form web yang diisi oleh pengguna dengan data seperti nama, email, jumlah paket, layanan, alamat, dan prioritas. 
Setelah dikirim, data akan diproses menggunakan PHP dengan melakukan pengecekan kelengkapan input. Sistem menentukan harga berdasarkan jenis layanan yang dipilih menggunakan array, 
lalu menghitung total biaya dari jumlah paket serta menambahkan biaya jika memilih prioritas tertentu. 
Selanjutnya, sistem menampilkan kembali data dan hasil perhitungan pada halaman konfirmasi agar pengguna dapat memastikan informasi yang telah dimasukkan.
